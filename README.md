## 완성된 기능

### 1. 헤더 및 로고
- **경영인증평가원 로고**: 블루 다이아몬드 형태의 심볼 로고
- **고정 헤더**: 스크롤 시에도 상단에 고정되는 네비게이션
- **브랜드 아이덴티티**: 깔끔한 타이포그래피와 영문 서브타이틀

### 2. 히어로 섹션
- **메인 캐치프레이즈**: "제조를 잇다"
- **서비스 소개**: 온라인 공장 플랫폼의 핵심 가치 전달
- **그라디언트 배경**: 블루 계열의 모던한 그라디언트

### 3. 공장 연결 서비스 개요
- **고객용 서비스 카드**: 부품제조, 제품개발, 완제품 위탁생산 소개
- **공장용 서비스 카드**: 온라인 영업 및 견적 수신 소개
- **아이콘 기반 UI**: 명확한 시각적 구분

### 4. 서비스 상세 섹션
#### 고객용 서비스 (Client)
- **부품 제조**: 정밀 가공, 금형, 주조, 프레스 등
- **제품 개발**: 아이디어부터 시제품까지
- **완제품 위탁생산**: OEM/ODM 생산 지원
- **온라인 견적 요청**: 도면/사양서 업로드 및 비교 견적

#### 공장용 서비스 (Factory)
- **온라인 영업**: 영업 인력 없이 고객 확보
- **공장 프로필 등록**: 설비, 기술력, 생산능력 홍보
- **견적 관리 시스템**: 체계적인 견적 관리 및 응답
- **인증 파트너 혜택**: 우선 노출 및 마케팅 혜택

### 5. CTA (Call to Action) 섹션
- **파트너 찾기 버튼**: 주요 행동 유도
- **견적 요청하기 버튼**: 서브 행동 유도
- **다크 배경**: 눈에 띄는 섹션 구분

### 6. 이용 절차
- **4단계 프로세스**: 회원가입 → 견적요청 → 견적수신 → 거래시작
- **비주얼 프로세스**: 단계별 숫자와 화살표로 시각화
- **명확한 가이드**: 사용자 여정 안내

### 7. 디자인 시스템
- **깔끔한 레이아웃**: 화이트 스페이스 활용
- **일관된 컬러**: 블루(고객), 오렌지(공장) 색상 체계
- **모던 타이포그래피**: 가독성 높은 폰트 사이즈
- **부드러운 애니메이션**: 호버 효과와 트랜지션
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

## 개발 환경 설정

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn
- Git

### 로컬 개발 시작
```bash
# 의존성 설치 (이미 완료됨)
npm install

# 빌드
npm run build

# 개발 서버 시작 (PM2)
pm2 start ecosystem.config.cjs

# 개발 서버 재시작
pm2 restart webapp

# 개발 서버 중지
pm2 stop webapp

# 로그 확인
pm2 logs webapp --nostream
```

### 스크립트
```bash
# 개발 서버 (Vite)
npm run dev

# 빌드
npm run build

# 프리뷰 (Wrangler)
npm run preview

# 배포
npm run deploy

# Cloudflare 타입 생성
npm run cf-typegen
```
