# 경영인증평가원

## 프로젝트 개요
- **이름**: 경영인증평가원 (Management Certification Institute)
- **목표**: 검증된 우수 제조업체와 고객을 연결하는 인증 플랫폼
- **핵심 기능**: 
  - 제조업체 인증 및 검증
  - 인증 업체 목록 제공
  - 온라인 견적 요청 시스템 (RFQ)

## 완성된 기능

### 1. 전문적인 인증기관 디자인
- **헤더 네비게이션**: 기관소개, 인증업체, 견적요청, 문의하기
- **로고 시스템**: 다이아몬드 형태의 블루 심볼
- **깔끔한 레이아웃**: 전문성과 신뢰성을 강조하는 디자인

### 2. 히어로 섹션
- **핵심 메시지**: "신뢰할 수 있는 제조 파트너"
- **CTA 버튼**: 견적 요청하기, 인증업체 보기
- **전문적인 톤앤매너**: 인증기관의 권위 있는 이미지

### 3. 기관 소개 섹션
- **엄격한 인증 기준**: 품질 관리, 생산 능력, 재무 건전성 평가
- **검증된 업체**: 인증 통과 업체만 등록
- **신뢰할 수 있는 거래**: 안전하고 투명한 거래 환경

### 4. 인증 업체 섹션
실제 인증받은 6개 업체 예시:
- **(주)정밀기계**: 정밀 가공, CNC 가공, 5축 가공
- **(주)금형산업**: 금형 제작, 사출 성형, 대량 생산
- **(주)판금기술**: 판금, 용접, 레이저 커팅
- **(주)표면처리**: 표면처리, 도금, 양극산화
- **(주)주조기술**: 주조, 단조, 알루미늄 주조
- **(주)조립제작**: 조립, 완제품, OEM/ODM

**업체 카드 기능:**
- 인증 배지 표시
- 업체명 및 전문 분야
- 주요 서비스 특징 (체크리스트)
- 견적 요청 버튼

### 5. 견적 요청 폼 (RFQ System)
CAPA.ai 스타일의 전문적인 견적 요청 시스템 구현

#### 제조 서비스 선택 (필수)
**제품개발/부품제조:**
- 원스톱 제품개발
- 기구설계
- 제품 디자인
- 3D프린팅
- 절삭(CNC)
- 판금
- 프레스
- 금형/사출 성형
- 주조
- 단조
- 용접
- 열처리
- 표면처리

**완제품/위탁생산:**
- 조립
- OEM 생산
- ODM 생산
- 포장

#### 상세 정보 섹션
- 수량 입력
- 희망 납기 (날짜 선택)
- 도면 및 사양서 첨부 (PDF, DWG, DXF, STP, STEP, 이미지)

#### 폼 기능

## 개발 환경 설정

### 로컬 개발
```bash
# 빌드
npm run build

# 개발 서버 시작 (PM2)
pm2 start ecosystem.config.cjs

# 서비스 재시작
pm2 restart webapp

# 로그 확인
pm2 logs webapp --nostream

# 서비스 중지
pm2 stop webapp
```

## 특징 및 장점

### 사용자 경험
- **직관적인 네비게이션**: 원페이지 구조로 쉬운 탐색
- **명확한 정보 전달**: 섹션별로 구분된 정보
